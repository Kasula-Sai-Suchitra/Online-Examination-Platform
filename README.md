# Online Examination & Proctoring Platform

## Overview

The Online Examination & Proctoring Platform is a secure web-based solution designed to conduct online assessments efficiently while maintaining exam integrity through automated monitoring and evaluation.

## Features

### Timed Examinations

* Configurable exam duration
* Automatic submission on timeout
* Real-time countdown timer

### Question Bank Management

* Multiple question categories
* Randomized question generation
* Support for multiple-choice questions

### Webcam-Based Proctoring

* Live webcam monitoring
* Suspicious activity detection
* Exam session recording

### Automated Evaluation

* Instant answer checking
* Automatic score calculation
* Performance reports

### Result Analytics

* Student performance dashboard
* Subject-wise analysis
* Score visualization and reporting

## Technology Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL

### Real-Time Communication

* WebRTC
